# Security Policy

## Private files

Do not commit:

- Discord Application IDs if you consider them private
- Discord client secrets
- bot tokens
- API keys
- `.env` files
- `DiscordPresence.local.json`
- licensed font files
- extracted game assets

## Reporting issues

Open a GitHub issue for normal bugs. For private/security-sensitive reports, contact the maintainer privately instead of posting secrets in an issue.
